Что такое окружение JavaScript?

Js изначально был создан сугубо для браузеров, но сегодня является мультиплатформенным языком программирования для решения широкого круга задач:

  • в веб браузере
  • в веб сервере
  • в любой другой среде, даже в настольных часах

Каждая такая среда предоставляет свою функциональность, которую спецификация js называет окружением, которое предоставляет свои объекты и дополнительные функции в дополнение к базовым возможностям js:

  • Браузеры предоставляют возможность для управления веб страницами
  • Node.js некоторые серверные функции

Нас будет интересовать только браузерное окружение.
