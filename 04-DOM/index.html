Объектная модель документа DOM

DOM представляет содержимое страницы в виде объектов, которые можно менять.

Основой любого html документа являются теги и в соответствии с DOM каждый тег является объектом:

  • вложенные теги являются дочерними и потомками своего родителя
  • текст и комментарии тоже объекты

Абсолютно все объекты доступны из js и мы можем ими управлять.

Как взаимодействовать с DOM?

Все взаимодействия начинаются с объекта document - это основная точка входа в дом. Например, Самые верхние элементы дерева доступны как свойства объекта document:

const htmlEl = document.documentElement
const headEl = document.head
const bodyEl = document.body
const firstChildNode = bodyEl.firstChild
