Объектная модель браузера (BOM)

BOM (объектная модель браузера) - дополнительные объекты, предоставляемые браузером для работы со всем, кроме документа.

Примеры использования BOM

Получение данных о браузере и ОС

console.log(navigator.userAgent);

Текущий адрес страницы

console.log(location.href);

Перенаправление пользователя

location.href = 'https://fls.guru';

Управление историей браузера

history.back();
history.forward();

Полезные методы для взаимодействия с пользователем

Существуют еще полезные методы, которые тоже часть BOM:

  • alert
  • confirm
  • prompt

Пример использования диалоговых окон

alert('hi');
const confirmResult = confirm('Хочешь верстать?');
console.log(confirmResult);
const promptResult = prompt('Who are you in life?');
console.log(promptResult);
